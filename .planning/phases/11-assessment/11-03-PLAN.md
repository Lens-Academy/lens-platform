**Part A: Add `getRoleplayAssessment()` to `web_frontend/src/api/roleplay.ts`**

Add a function that fetches the assessment for a completed session:
```typescript
export async function getRoleplayAssessment(
  sessionId: number,
): Promise<{
  score_data: {
    overall_score: number;
    reasoning: string;
    dimensions?: Array<{ name: string; score: number; note?: string }>;
    key_observations?: string[];
  };
  model_id: string | null;
  created_at: string;
} | null> {
  const res = await fetchWithRefresh(
    `${API_BASE}/api/chat/roleplay/${sessionId}/assessment`,
    {
      credentials: "include",
      headers: { "X-Anonymous-Token": getAnonymousToken() },
    },
  );

  if (res.status === 404) return null;
  if (!res.ok) throw new Error("Failed to fetch assessment");
  return res.json();
}
```

**Part B: Add assessment score card to RoleplaySection**

After the roleplay is completed (`isCompleted === true`), the component should:

1. If the segment has `assessmentInstructions`, start polling for the assessment result using `getRoleplayAssessment(sessionId)`. Poll every 2 seconds, up to 30 seconds (15 attempts). Use a `useEffect` triggered by `isCompleted && sessionId && segment.assessmentInstructions`.

2. Track assessment state: `assessmentState: "loading" | "ready" | "unavailable"` and `assessmentData`.

3. In the "Done state" area (where "Conversation complete" and "Try again" button are), add a score card above the retry button:
   - **Loading state**: "Assessing your conversation..." with a subtle spinner/pulse animation
   - **Ready state**: A card showing:
     - Overall score as "X/5" with the reasoning text below
     - If `dimensions` exist, show each as a small labeled score
     - If `key_observations` exist, show as a bulleted list
     - A "Discuss your performance" button (indigo outline style, consistent with roleplay theme)
   - **Unavailable state** (no assessment-instructions or timeout): Just show "Conversation complete" as before

4. The "Discuss your performance" button should call a new `onFeedbackTrigger` callback prop (to be wired in Module.tsx). Pass the formatted assessment data as a string that will seed the feedback chat.

Add `onFeedbackTrigger?: (assessmentSummary: string) => void` prop to RoleplaySection.

The assessment summary string format for seeding the feedback chat:
```
I just completed a roleplay exercise. Here are my results:

Score: X/5
Reasoning: [reasoning from assessment]
Key observations:
- [observation 1]
- [observation 2]

Can you help me understand how to improve?
```
